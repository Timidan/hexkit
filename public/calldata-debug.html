Test Calldata Updates

This page helps debug the calldata updating issue.

Test Cases:

  1. Function Selection → Input Change → Calldata Update
    • Select a function from the dropdown
    • Change input values
    • Verify calldata updates in real-time
  2. Network/Contract Change → Calldata Reset
    • Change network or contract
    • Verify calldata resets to "0x"
    • Verify no stale calldata remains
  3. Input Format Edge Cases
    • Empty inputs
    • Invalid inputs for type
    • Special characters in inputs
